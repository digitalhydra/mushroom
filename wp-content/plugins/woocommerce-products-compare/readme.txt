Allow customers to compare similar products side by side

See http://docs.woothemes.com/document/woocommerce-products-compare/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-products-compare' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
